Integration Notes

Important Notes
  • Tab Service is Singleton: TabStateService is registered as Singleton to persist tab state across page navigations
  • Integration with CodeSetManager: CodeSetTabs can be integrated into CodeSetManager.razor by calling TabStateService.OpenTab(codeSet)
  • No Duplicate Tabs: Opening the same CodeSet twice will activate the existing tab
  • Dirty State: Editing items, adding, or deleting will mark the tab as dirty
  • Pinned Tabs: Pinned tabs are protected from Close All and Close Others operations
  • Future Enhancement: Consider adding localStorage persistence for tab state

Section 1: Manual Testing Steps

Minimap Component

  • Minimap displays in bottom-right corner
    Position: absolute, bottom: 16px, right: 16px
  • Minimap shows all workflow nodes
    Each node type has correct color (green/indigo/orange/gray)
  • Node colors match type correctly
    Start=green, Step=indigo, Decision=orange, Action=purple, End=gray
  • Viewport indicator shows current view area
    Blue border with semi-transparent fill
  • Viewport updates when panning canvas
    Position changes to reflect current view
  • Viewport size changes with zoom level
    Larger viewport at lower zoom, smaller at higher zoom
  • Clicking minimap navigates to that location
    Canvas centers on click position
  • Collapse/expand toggle works
    Chevron icon changes direction, content hides/shows

Zoom Controls

  • Zoom controls display in bottom-left corner
    Vertical stack with +, %, -, divider, fit, reset buttons
  • Zoom in button increases zoom by 10%
    Click + button, zoom increases
  • Zoom out button decreases zoom by 10%
    Click - button, zoom decreases
  • Zoom level displays correctly (e.g., "100%")
    Shows rounded percentage
  • Zoom in disabled at max (200%)
    Button appears disabled, click has no effect
  • Zoom out disabled at min (25%)
    Button appears disabled, click has no effect
  • Fit to view button works
    Calculates zoom to show all nodes with padding
  • Reset view button works
    Returns to zoom=1, pan=(0,0)

Fit Controls

  • Fit controls display in top-right corner
    Horizontal row of buttons
  • Fit View button shows all nodes
    Adjusts zoom and pan to fit all content
  • Center button centers canvas
    Moves view to center of canvas
  • Reset button restores default view
    Zoom=100%, position=(0,0)

Pan and Zoom Interactions

  • Mouse wheel zooms canvas
    Scroll up = zoom in, scroll down = zoom out
  • Space + drag pans canvas
    Hold space, then click and drag to move view
  • Middle mouse button pans canvas
    Click middle mouse and drag to pan
  • Pan indicator shows during pan
    Semi-transparent overlay with "Panning" text
  • Pan indicator hides when pan ends
    Fades out on mouseup

Keyboard Shortcuts

  • Keyboard hints display at bottom
    Shows available shortcuts
  • Ctrl + 0 resets view
    Same as Reset button
  • Ctrl + 1 fits view
    Same as Fit View button

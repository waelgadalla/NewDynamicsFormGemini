Section 1: Manual Testing Steps

Path Type Rendering

  • Bezier curves render with smooth S-curve
    Control points at 50% distance create smooth curves
  • Step paths render with 90-degree angles
    Horizontal segment at midpoint between nodes
  • Smooth Step paths render with rounded corners
    10px radius on corners, uses quadratic bezier
  • Straight lines render directly between handles
    Simple line from source to target

Arrow Markers

  • Arrow heads point toward target node
    marker-end attribute with orient="auto"
  • Default arrows are gray (#6B7280)
    Uses arrow-default marker
  • Success arrows are green (#10B981)
    For Yes branch connections
  • Warning arrows are orange (#F59E0B)
    For No branch connections
  • Active arrows are indigo (#6366F1)
    For selected/hovered connections

Selection and Hover States

  • Clicking connection selects it
    OnSelected event fires with connection ID
  • Selected connection has thicker stroke (3px)
    Stroke changes to indigo active color
  • Hover highlights connection path
    Same styling as selected state
  • Cursor changes to pointer on connection
    pointer-events: stroke enables click detection

Delete Button

  • Delete button appears on selection
    Red circular button with X icon
  • Delete button appears on hover
    Same positioning as selection
  • Delete button positioned at path midpoint
    Calculated from source and target positions
  • Clicking delete removes connection
    OnDeleted event fires with connection ID
  • Delete button scales on hover (1.1x)
    Visual feedback for interaction

Drawing Preview

  • Drawing shows animated dashed line
    stroke-dasharray: 5 with animation
  • Preview line follows mouse position
    UpdateDrawing() called on mousemove
  • Preview uses active arrow marker
    Indigo arrow at end of drawing line
  • Preview disappears on cancel
    CancelDrawing() resets state

Branch Labels

  • Labels display for connections with Label property
    Only shows when Label is not empty
  • Labels positioned at 25% of path
    Closer to source node
  • Yes labels show green text
    BranchType.Yes styling
  • No labels show orange text
    BranchType.No styling
  • Labels have white background with border
    rect element behind text

Connection Creation

  • StartDrawing initializes drawing state
    Sets IsDrawing, source node, and handle
  • FinishDrawing creates new connection
    OnConnectionCreated event fires
  • Cannot connect node to itself
    sourceNodeId != targetNodeId check
  • Connection includes handle positions
    SourceHandle and TargetHandle properties

Usage Example

Example of how to use the ConnectionLayer component in a parent WorkflowCanvas:

@* In WorkflowCanvas.razor *@ @using VisualEditorOpus.Components.Workflow <div class="workflow-canvas"> <ConnectionLayer Connections="@_connections" NodePositions="@_nodePositions" PathType="@ConnectionType.Bezier" SelectedConnectionId="@_selectedConnectionId" OnConnectionSelected="HandleConnectionSelected" OnConnectionCreated="HandleConnectionCreated" OnConnectionDeleted="HandleConnectionDeleted" @ref="_connectionLayer" /> @* Nodes rendered here *@ </div> @code { private ConnectionLayer? _connectionLayer; private List<WorkflowConnection> _connections = new(); private Dictionary<string, NodePosition> _nodePositions = new(); // Call from node handle mousedown private void StartConnectionDrawing(string nodeId, HandlePosition handle) { var pos = GetHandleScreenPosition(nodeId, handle); _connectionLayer?.StartDrawing(nodeId, handle, pos); } }

Rule Types Overview

Skip Rules

Skip a specific step/module when conditions are met. Example: Skip manager approval for amounts under $1000.

Go To Rules

Navigate to a specific step when conditions are met. Example: Return to personal info if validation fails.

Completion Rules

Complete the workflow when conditions are met. Example: Auto-complete workflow for pre-approved requests.

Validation Rules

Custom field validation based on conditions. Example: Require additional documents for high-value transactions.

Usage Example

Example of how to use the WorkflowRulesModal component:

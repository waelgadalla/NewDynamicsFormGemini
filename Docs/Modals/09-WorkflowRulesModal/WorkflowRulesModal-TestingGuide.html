  • 2. ConditionBuilderModal Integration
    Ensure ConditionBuilderModal is implemented and accepts AvailableFields with cross-module references

    Parent component must pass the list of FormModuleSchema objects for step titles and field references
  • 4. Rule Execution in Workflow Engine
    Implement the actual rule evaluation and execution logic in the workflow runtime engine

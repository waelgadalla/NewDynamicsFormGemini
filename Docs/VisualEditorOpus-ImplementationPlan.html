Valuable Patterns from DynamicForms.Editor

The following patterns and features from DynamicForms.Editor's EditorStateService should be incorporated into VisualEditorOpus:

Key Patterns to Adopt

  • JS Interop Keyboard Shortcuts - [JSInvokable] methods for Ctrl+S, Ctrl+Z, Ctrl+Y, Del, Ctrl+C, Ctrl+V, Ctrl+D, Escape
  • Toast Feedback - Inject IToastService and provide user feedback for every action
  • Clipboard to System - Copy field JSON to system clipboard for cross-window paste
  • Import/Export Methods - ExportModuleJsonAsync, ExportWorkflowJsonAsync with file download
  • Workflow Layout Persistence - Save node positions in ExtendedProperties as JSON
  • Circular Reference Check - HasCircularReference() helper method
  • State Suppression - suppressStateChanged parameter to batch updates
  • Bilingual Labels - "Copy of" / "Copie de" pattern for duplication

Circular Reference Detection private bool HasCircularReference(FormFieldSchema field, FormModuleSchema module) { var visited = new HashSet<string>(); var current = field; while (current?.ParentId is not null) { if (!visited.Add(current.Id)) return true; current = module.Fields.FirstOrDefault(f => f.Id == current.ParentId); } return false; }

Manual Implementation Tasks

Add to RightSidebar

Add the TypeConfigButton component to the property panel for fields that support TypeConfig.

Components/Editor/RightSidebar.razor

Add after GeneralSection:

@if (HasTypeConfig) { <TypeConfigButton /> }

Add Imports for ConfigPanels

Ensure the ConfigPanels folder is imported in _Imports.razor

Components/_Imports.razor

Add:

@using VisualEditorOpus.Components.Properties.Sections.ConfigPanels

Include ConfigPanels CSS

The ConfigPanels.razor.css file needs to be available. Consider adding to App.razor or using scoped CSS.

May need manual CSS import

Additional Field Types

Time, Image, Signature, and AutoComplete fields need their own config panels implemented.
